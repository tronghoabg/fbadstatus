FB AD Status

Tool to check advertising account information quickly and conveniently.

Track information of all your ad accounts

Fbadstatus provides a summary of all the information of those ad accounts such as: Balance, threshold, limit/day, total spend, payment date, ... without you needing to click on each account. ads to check as before. This tool also supports users to calculate the amount to spend each day or each time that the user needs.

Track business information

FbAdStaus provides an overview of all information of that Business account (BM) such as: BM name, role, Limit BM, etc. without you having to click on each BM to check like before.

View Fanpage status

View all detailed information of the fan page under management

Share pixels with multiple Ad Accounts

Fbadstatus activates the hidden feature from Facebook. It will help users to use a personal account with a pixel attached to BM.

How to Install a Chrome Extension in Developer Mode

Step 1 - download a zip file with the extension

If you already have downloaded file - skip this step. If you don't have a file, click to download in the top site.

Step 2 - open the extension page in google chrome

There are several ways todo that.
Option 1: type chrome://extensions in the url bar and press enter.
Option 2: click on the tree dots in the top right of the browser, then click "More tools -> Extensions"
Option 3: click on the tree dots in the top right of the browser, "setting" > "Extension".

Step 3 - activate developer mode

Turn on the switch on the top right of the page that says "Developer mode";.

Step 4 - load unpacked extension

Option 1: Drop file zip to install extension

Option 2: extract the contents of the zip file and load unpacked extension
